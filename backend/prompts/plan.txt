## 사용자 요청
{user_prompt}

## 최근 러닝 활동 (최대 3건)
{recent_activities}

## 다가오는 대회
{upcoming_races}

## 사용자 정보
- 성별: {gender}
- 출생: {birth_year}년 {birth_month}월
- 오늘 날짜: {today}

## 작성 규칙
- 아마추어 러너 대상이므로 과도하게 자잘한 계획은 지양
- 특별한 요청이 없으면 1주(7일) 계획 작성
- 세션 유형: Easy, Long, Interval, Fast, Recovery, Rest, Race
- 날짜는 {today} 이후로 설정
- 목표 거리는 미터 단위, 목표 페이스는 초/km 단위

## 출력 형식
반드시 아래 JSON 형식으로만 응답하세요. JSON 외의 텍스트를 포함하지 마세요.

```json
{{
  "plan_text": "계획 전체를 설명하는 한국어 텍스트 (300자 이내)",
  "sessions": [
    {{
      "date": "YYYY-MM-DD",
      "session_type": "Easy|Long|Interval|Fast|Recovery|Rest|Race",
      "title": "세션 제목 (예: 가벼운 조깅, 인터벌 5x1km)",
      "description": "세션 설명 (WU/CD 포함, 구체적 지시)",
      "target_distance": 5000,
      "target_pace": 360
    }}
  ]
}}
```
